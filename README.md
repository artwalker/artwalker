## Culprit

Culprit is built for engineering and product teams that want clearer visibility into what is actually happening across Jira.

It can help with:

- Delivery-risk analysis
- Sprint and backlog summaries
- Bottleneck detection
- Issue-quality analysis
- Release-readiness reports
- Automated engineering status updates
- Custom workflow intelligence for teams

The goal is simple: help teams see risks earlier, reduce manual reporting work, and make engineering execution easier to understand.

## Sonechka

Culprit is powered by **Sonechka**, my self-developed AI agent engine.

Sonechka is focused on:

- Reliable long-running task execution
- Retrieval and evidence handling
- Memory and continuity
- Workflow automation
- Auditability and traceability
- Tool use with visible boundaries
- TUI and Telegram interaction surfaces

Sonechka is not a thin LLM wrapper. It is an agent runtime designed to make AI automation more useful, inspectable, and trustworthy.
